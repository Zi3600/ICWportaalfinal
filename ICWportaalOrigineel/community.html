HACKATHON V1

De zomercompetitie van 2026

Bouw deze zomer een site over zonnepanelen. Maak iets dat mensen daadwerkelijk willen kopen.

Eén thema, één maand, één winnaar.

2000 V-Bucks
OF EEN EQUIVALENT IN CASH

De opdracht

Maak een site over zonnepanelen die mensen verkoopt. Niet alleen mooi van buiten — je moet ook één creatieve functie inbouwen die mensen helpt de keuze te maken om er een te kopen.

Denk aan een rekenmachine die laat zien hoeveel je bespaart, een vergelijker, een visualisatie op je dak — als het overtuigt, telt het.

  • Thema: zonnepanelen — verkoop het idee
  • Stack: HTML, CSS, JavaScript (alles wat je in het leerpad gezien hebt)
  • Creatieve functie: één feature die overtuigt, verplicht
  • Style: wordt mee beoordeeld, dus probeer iets dat niet als template eruit ziet
  • AI: mag je zeker gebruiken (we gebruikten het zelfs voor deze site aan te maken so ur good), wat je gemaakt hebt met AI moet je kunnen uitleggen wat jouw functie doet als je het word gevraagd

Hoe de competitie verloopt

1

Bouwfase

JULI — AUGUSTUS 2026

Je hebt een hele maand om je site te bouwen. Werk solo, duo of in groep. Lever in via GitHub repo door te storen naar de mail: hackathon@icw.xyz.

2

Showcase call

BEGIN SEPTEMBER 2026

Op call laat je je site zien en leg je uit wat je gebouwd hebt. Vragen worden gesteld over je code, je moet weten wat er staat. (Again je mag AI gebruiken, maar je moet de volledige code kunnen uitleggen)

3

Finale: 1u live build

FINALISTEN ONLY

Haal je de finale? Dan krijg je één uur om iets te bouwen met enkel HTML en CSS uit het leerpad. Geen JavaScript, geen externe hulp. Pure skill check.

Spelregels

AI mag

Gebruik Ai enz, boeit de competitie vrij weinig. Maar als je niet kan uitleggen wat je code doet, lig je eruit.

Eigen werk

Templates kopiëren zonder bronvermelding is geen optie. Inspiratie nemen wel. wees gewoon eerlijk over je bronnen: bij inspiratie en bronvermelding krijg je bonuspunten.
